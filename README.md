Ammar Ajmal's portfolio is a modern, full-stack web application built with HTML, CSS, Bootstrap, Firebase, and EmailJS. It serves as a professional showcase for Ammar, a Software Engineering graduate from Rohela Tejeka, Okara, featuring a warm and premium aesthetic with an ivory, charcoal, and gold color scheme, glass morphism effects, and smooth AOS scroll animations. The website includes multiple sections: a dynamic home page with an image collage, a personal story presented with programming syntax, a projects section powered by Firebase Firestore, social media cards with preview images, and an elegant contact form that captures the user's name, email, subject, and message. A privacy section handles cookie consent with geo-location tracking, recording browser details, region, and coordinates upon permission. The admin panel is hidden and accessible only via a secret footer link, requiring Firebase email/password authentication every single time for security. Once logged in, the admin can view total visits, query statistics, detailed logs of all inquiries with full message history, recent visitor locations, and respond to pending queries. The email system uses EmailJS to send messages from the user's address to immianammar@gmail.com via the contact form, and allows the admin to reply from immianammar@gmail.com directly to the client's email. All data is stored securely in Firebase Firestore, including visits, queries, and projects. The website is fully responsive, works across all devices, and can be easily deployed to platforms like Netlify or Vercel. Configuration is straightforward—developers need to set up Firebase Authentication, Firestore, and EmailJS with their own credentials, update the security rules, and optionally add sample project data. The code is contained in a single HTML file, making it simple to customize colors, social links, and story text. This portfolio combines elegant design with powerful functionality, offering a complete solution for professionals who want to showcase their work, track engagement, and manage inquiries securely.
